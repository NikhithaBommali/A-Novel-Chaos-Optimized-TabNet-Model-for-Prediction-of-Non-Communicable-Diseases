# Branch Workflow Brief

## Base branch for future work

The current selected base branch for this repository is `main`.

This is the correct base branch to use for future work planning.

All future implementation branches for both API and UI work must be created from `main`.

This request is planning/process guidance only. No immediate code, configuration, or branch changes are required.

## Pre-work validation

Before starting work, contributors can manually confirm they are on or branching from `main` with:

```sh
git branch --show-current
```

If needed, also check the repository status and the branch ancestry before creating a new branch.

## Manual verification checklist

- Confirm the repository base branch is `main`.
- Confirm the planned API or UI work will start from `main`.
- Confirm no code, config, or branch updates are required for this documentation-only guidance.
- Confirm the pre-work validation command shows `main` when run from the intended starting point.
